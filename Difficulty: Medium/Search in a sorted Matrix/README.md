<strong>Explanation</strong>: 62 is not present in the matrix, so output is 0.
</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input</strong>: mat[][] = [[18, 21, 27, 38, 55, 67]], x = 55<br><strong>Output</strong>: true
